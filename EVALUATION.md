# Evaluation and Metrics

This document provides a measurable evaluation of the Precision-Aware Type Demotion compiler tool.

## 1. Metrics & Success Criteria
The success of this compiler optimization is measured along two primary axes:
1. **Memory Savings:** The percentage of 32-bit floating-point or 32-bit integer variables that were successfully and safely demoted to 16-bit or 8-bit representations. Formula: `(Demoted_Vars / Total_Vars) * 100%`.
2. **Absolute Error Bound:** The numerical accuracy loss induced by the precision reduction must not exceed the target threshold defined by the developer (default: $E = 0.05$).

## 2. Baseline Comparison
The baseline is defined as the unmodified C source code compiled natively with `gcc -O3`, utilizing standard IEEE-754 32-bit `float` representations.

The optimized executable is defined as the modified C code generated by our Clang tool, compiled natively using the same `gcc -O3` parameters, utilizing IEEE-754 16-bit `_Float16` representations where flagged as safe.

## 4. Failure Case / Boundary Demonstration

To prove the safety of the compiler engine, we implemented two extreme boundary test cases where the compiler **must** fail the demotion intentionally to prevent data corruption.

### Test Case 6: Extreme Integer Bounds (`kernel_int_extreme.c`)
- **Scenario:** A loop bound of `100,000,000` is defined, exceeding `int16_t` max value (`32,767`).
- **Compiler Action:** The Precision Analysis engine correctly identifies the overflow risk and raises a `[WARNING]`.
- **Result:** Demotion is aborted. Variables are safely retained as `int32_t`. -> **✅ SAFELY ABORTED**

### Test Case 7: Extreme Double Precision (`kernel_double_extreme.c`)
- **Scenario:** Adds a tiny delta (`1.0e-15`) to a base value. Demoting to FP32 would cause catastrophic cancellation (the delta is completely lost).
- **Compiler Action:** The engine simulates the FP32 conversion, calculates the error against the FP64 baseline, and sees it exceeds the allowable loss. It raises a `[FATAL]` error.
- **Result:** Optimization is safely aborted. Original FP64 code is retained. -> **✅ SAFELY ABORTED**
